# IndexFlow Frontend

Demo dashboard for **IndexFlow**, visualizing deterministic usage verification, insights, and AI commentary.

## What this repo contains

- Campaign-based demo dashboard
- Score and farming risk visualization
- Wallet-level drilldowns with proof hash
- Optional AI commentary display (non-proof)
- Explicit data source indicators (run / insights / commentary)

## Purpose

This frontend is intended for:
- Product demos
- Events and investor presentations
- Early partner feedback

It is **not** a production UI.

- Demo-ready
